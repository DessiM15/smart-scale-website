# Services Component

Service card components with slider functionality inspired by Auxa template.

## Features

- Swiper.js slider for horizontal scrolling
- Large numbered service cards (.01, .02, etc.)
- Image overlays with hover effects
- "Learn More" links with icons
- Background image layers
- Smooth transitions

## Structure

```
services/
├── service-slider.html  # Service slider HTML structure
└── styles.css          # Service-specific styles
```

## Integration Notes

- Adapt for Smart Scale's 4 services
- Use current service content
- Maintain current service images
- Connect to services.html page
- Ensure accessibility for screen readers
