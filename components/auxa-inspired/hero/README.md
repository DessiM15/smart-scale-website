# Hero Component

Hero section with animations and parallax effects inspired by Auxa template.

## Features

- Large, bold typography with gradient text effects
- Background image with parallax
- Numbered stat counters
- Call-to-action button with hover effects
- Multiple image layers for depth
- Scroll animations (WOW.js)

## Structure

```
hero/
├── hero-section.html    # Hero HTML structure
└── styles.css          # Hero-specific styles
```
