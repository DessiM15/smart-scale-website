# Header Component

Navigation and header structure inspired by Auxa template.

## Features

- Sticky header with scroll behavior
- Mobile-responsive hamburger menu
- Overlay menu with slide-in animation
- "Get In Touch" CTA button
- Clean, minimal design

## Structure

```
header/
├── navigation.html      # Main navigation HTML structure
├── mobile-menu.html     # Mobile menu overlay structure
└── styles.css          # Header-specific styles
```
