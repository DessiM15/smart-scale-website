# Footer Component

Footer design inspired by Auxa template.

## Features

- Multi-column layout (4 columns)
- Newsletter subscription form
- Social media links
- Footer links with hover effects
- Copyright section
- Responsive design

## Structure

```
footer/
├── footer.html         # Footer HTML structure
└── styles.css         # Footer-specific styles
```

## Integration Notes

- Adapt for Smart Scale's footer content
- Keep current contact information
- Maintain current social links
- Add newsletter form if needed
- Update copyright information

## Footer Sections

1. **Column 1:** Logo, company description, social links
2. **Column 2:** Quick links (About, Services, etc.)
3. **Column 3:** Additional links (Portfolio, Contact, etc.)
4. **Column 4:** Newsletter signup, contact info
